# 📈 NVIDIA Stock Price Prediction using Machine Learning

## 🚀 Overview

This project focuses on predicting the stock price of **NVIDIA (NVDA)** using historical market data and machine learning techniques. The model analyzes past trends, patterns, and indicators to forecast future stock prices.


---

## 🧠 Project Objectives

* Collect and preprocess historical stock data
* Perform exploratory data analysis (EDA)
* Build predictive models using machine learning
* Evaluate model performance
* Visualize actual vs predicted stock prices

## 🤖 Models Used

* Linear Regression
* Random Forest
* LSTM (for time series prediction)

---

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

---

## 📉 Results

* The model predicts stock price trends based on historical data
* Performance improves with feature engineering and tuning
* LSTM performs better for sequential data
